Content

This semester it is all about Go.

We will answer the following questions.

  • What makes Go different from Java?

  • How are Go features such as method overloading and generics translated?

  • What testing methods are offered for Go?

Lecture material

  • Go basics

  • Go methods, interfaces, generics

  • Further examples

  • Translating Go

  • Generic Go limitations

  • Go fuzz testing

Further topics that we might explore (in some projects).

  • Static versus dynamic typing
  • Type inference
  • Concurrency in Go
  • Fuzzing concurrent Go
