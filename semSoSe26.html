Content

This semester it is all about Go, Rust and Haskell

We will answer the following questions.

  • Why do we need different programming languages?

  • What makes Go different from Java?

  • What testing methods are available?

  • What is the differences between testing and static verification?

  • Why do I care about programming given that there’s ChatGPT, …?

  • …

Lecture material

  • Go:
    • Go basics
    • Go methods, interfaces, generics
    • Further examples
  • Rust
    • Rust and stack-based VM + operational semantics rules
  • Haskell
    • Haskell

Specific topics

  • Testing
    • Go testing (user + unit tests, QuickCheck)
    • Go fuzz testing
  • Types
    • Expressive types
  • Programming language implementation
    • Stack-based VM and operational semantics rules (see Rust notes above)
    • Translating Go
    • Dictionary translation for Rust and Haskell (see Rust and Haskell notes above)
    • Haskell to implement other languages
  • Programming language comparison
    • Generic Go limitations
    • OO in Haskell and Go versus Haskell
    • Rust versus Haskell
